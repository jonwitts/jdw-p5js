Changing strokeweight on diagonals in a grid with colors

Mouse position x - left diagonal strokeweight, position y - right diagonal strokeweight; Left click - new random layout

Keys: s: save png, 1: round strokecap, 2: square strokecap, 3: project strokecap, 4: colour left diagonal, 5: colour right diagonal, 6: transparency left diagonal, 7: transparency right diagonal, 0: default
